
## 📌 MASTER PROJECT PROMPT

**Title:** Crime Vulnerability Analysis

### 1. Project Objective

You are tasked with building a **research-grade Exploratory Data Analysis (EDA) and Visualization project** to analyze **crime rates across Indian metropolitan cities** over **three years**, focusing on **vulnerable population groups**.

The project must:

* Perform **statistically sound EDA**
* Answer **research-oriented questions**
* Build a **Crime Vulnerability Index (CVI)**
* Rank **vulnerable groups** and **cities**
* Present results via a **modern, minimalist React dashboard**
* Be suitable for **academic publication**

---

### 2. Data Description

Assume access to a **master dataset** containing:

* `City` (Indian metropolitan cities)
* `Year` (3 consecutive years)
* `Vulnerable_Group`

  * Women
  * SC
  * ST
  * Children
  * Senior Citizens
* `Crime_Type`
* `Reported_Cases`
* `Population_of_Group`
* `Latitude`, `Longitude` (or city centroid)
* Any additional socio-legal indicators if available

---

### 4. Step-by-Step Analytical Workflow

## Step 1: Data Sanity Checks (Pre-EDA)

Perform and visualize:

* Missing value analysis
* Duplicate records
* Data type validation
* Outlier detection (IQR / Z-score)
* Year-wise consistency checks
* City-group population sanity validation

Deliverables:

* Python scripts
* React dashboard page: **“Data Health Overview”**

---

## Step 2: Core Exploratory Data Analysis (Research-Driven)

### Curate & Answer These Research Questions:

1. How does crime distribution vary **city-wise** and **year-wise**?
2. Which **vulnerable groups** experience consistently higher crime exposure?
3. Are crime trends **increasing, decreasing, or stable** across cities?
4. Which cities show **disproportionate crime rates** relative to population?
5. Are certain vulnerable groups systematically affected across metros?
6. Which crime types dominate per group and per city?

### Analyses to Perform:

* Trend analysis (YoY growth rates)
* City-wise comparison
* Group-wise crime intensity
* Per-capita crime normalization
* Variance & volatility analysis

Deliverables:

* Line charts (trend)
* Grouped bar charts
* Radar charts (group comparison)
* Insight annotations for research paper usage

---

Deliverables:

* Python geo-processing

## Step 4: Crime Vulnerability Index (CVI) Construction

### 4.1 Data Normalization

* Normalize crime metrics using:

  * Min-Max Scaling
  * Z-Score (justify choice)
* Adjust for population size

### 4.2 Vulnerable Group Weighting (Mathematically Justified)

Apply weights using:

* Severity of crime
* Legal & policy emphasis
* Historical discrimination
* Societal impact

**Proposed Weight Priority (Justify with math + literature):**

1. Women
2. SC
3. ST
4. Children
5. Senior Citizens

Use:

* Weighted Linear Combination
* Sensitivity analysis
* Explainable weighting logic

### CVI Formula (Example Structure):

```
CVI_city = Σ (Normalized Crime Rate_group × Group Weight)
```
