# FC-MT-LSTM Crime Prediction Dashboard

This React dashboard visualizes the results of the FC-MT-LSTM model for crime prediction with fairness constraints.

## Features

- Model performance comparison (MAE, RMSE, R²)
- Fairness gap analysis across protected groups (SC, ST, Women, Children)
- Geographic distribution of crime rates
- Real-time predictions interface
- Interactive charts and visualizations

## Setup Instructions

1. Install dependencies:
   ```bash
   cd dashboard
   npm install
   ```

2. Start the development server:
   ```bash
   npm start
   ```

3. The dashboard will be available at `http://localhost:3000`

## Data Integration

The dashboard loads data from:
- `../results/react_dashboard_data.json` (generated by `export_results_for_react.py`)

Make sure you have run the export script to generate the required dashboard data.

## Project Structure

- `public/` - Static assets
- `src/` - Main React source code
- `src/components/` - Reusable React components
- `src/App.js` - Main application component
- `src/App.css` - Application-wide styles

## Technologies Used

- React 18
- Material UI for styling
- Recharts for data visualization
- Axios for data fetching (planned for future integration)

## Development

To add new features or modify the dashboard:

1. Edit components in `src/` folder
2. Use Material UI components for consistent styling
3. Use Recharts for data visualization
4. Follow React best practices for component design
